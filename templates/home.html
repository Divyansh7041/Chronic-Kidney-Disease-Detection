Chronic Kidney Disease Detection

Predicting chronic kidney disease using machine learning for better early diagnosis and patient care.

Random Forest
Regression & Classification

Random Forest is a supervised learning algorithm that builds multiple decision trees and merges them to improve accuracy and stability.

XGBoost
Gradient Boosting Machine

XGBoost is a high-performance algorithm based on gradient boosting. It excels in structured data and is widely used in ML competitions.

Dataset

Used Kaggle dataset with 400 records and 25 features relevant to CKD prediction. Applied preprocessing and analysis to prepare the data.

Algorithm

Random Forest & XGBoost were used for accurate classification. Ensemble methods improved predictive performance.

Accuracy

Achieved an impressive accuracy of 98% using optimized feature selection and fine-tuned parameters.

How it Works

This prediction model is suitable for integration into healthcare applications. It enables early detection by analyzing medical parameters. The project is deployed using Flask to allow real-time predictions through a user-friendly interface.

Challenges Faced

Some challenges included dataset inconsistencies, environment setup issues, and library compatibility (especially with Python and TensorFlow). These were resolved using clear documentation and version control.

Future Enhancements

As a solo developer, I plan to enhance this project by expanding prediction to other diseases, improving UI/UX, and deploying on cloud platforms with database support for better accessibility and scalability.
